# Prizma Studio

**Автор:** MurKasKaa · **Версия:** 1.0.0

Объединённое десктопное приложение (Windows) для работы с PDF и PSD в одном окне.

## Быстрый старт

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python make_icon.py    # один раз, создаёт assets/icon.ico
python main.py         # dev-запуск
